A simple and lightweight pageview counter for your WordPress posts and pages. Compatible with all cache plugins.

Disclosure: After eanbled the plugin, each post/page will enguage a script to use AJAX technology to fetch and update counter from hithot.org. The service is free and public. The post ID will be used to distinguish the counter.

== Privacy ==

The counter data is stored in https://www.hithot.org. HitHot.org only uses site url and post ID to distinguish different post and log the counter.
